<h1> SIMPLE SOFT 3D </h1>

<p>本项目主要使用C++完成了OpenGL大部分矩阵和向量操作，为这方面的计算提供便利。同时附带两个小demo以供展示。矩阵向量算法部分代码应该可以兼容各个支持C++的编译器（iOS下可以mm文件后缀方式混用以及安卓下使用NDK编译）。</p>

<p>部分代码来参考自OpenGL.org(部分矩阵运算代码)，文件中有标注</p>
